# Veo3 Image Generation Mode — Settings Reference

From `settings_menu_image_mode.html` and user-provided details.

## Settings menu (Hình ảnh tab)

| Setting | Options | Notes |
|---------|---------|-------|
| **Tab** | Hình ảnh / Video | Image vs Video generation |
| **Aspect** | 16:9 (landscape) / 9:16 (portrait) | Same as video |
| **Multiplier** | x1, x2, x3, x4 | Number of images per prompt |
| **Model** | Nano Banana Pro, Nano Banana 2, Imagen 4 | Image models (different from video) |

## Image resolution (download)

When downloading generated images (context menu / edit page), resolution options are:
- **1k**
- **2k**
- **4k**

(Different from video: 720p, 1080p, 4k)

## Image upload folder

- **Label**: "Thư mục ảnh, ảnh mẫu phải được đặt tên là sample.png hoặc sample.jpg"
- **Required filename**: `sample.png` or `sample.jpg` (single reference image per project/prompt)
